# DigiShop Flutter App

DigiShop is a Flutter-based local shop discovery application developed as an MCA project. The app helps users explore nearby shops by category, view shop details, contact shop owners, and open shop locations in Google Maps.

## Features

- User authentication using Firebase Auth
- Store user profile data with Firestore
- Browse shops by category
- View shop details in a bottom sheet
- Call shop owners directly
- Open shop location in Google Maps
- Edit profile screen
- Clean and modern UI
- Firebase-based backend integration

## Categories Included

- Restaurants
- Clothes
- Hotels
- Shoes
- Mobile Shops
- Grocery Shops

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- URL Launcher
- Google Maps integration
